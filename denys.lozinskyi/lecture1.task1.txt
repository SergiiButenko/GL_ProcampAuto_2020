ЗАДАЧА:
"""
После неудачного мержа вам нужно удалить последний коммит в вашей ветке.
условие1. Ветка уже была отправлена в удаленный репозиторий с коммитом, который вам не нужен.
условие2. push -f отключен
"""


ВАРИАНТ РЕШЕНИЯ:
С помощью git revert.

1. git checkout моя_ветка
2. Смотрим дерево коммитов:
    git log
3. Копируем id коммита, который нам не нужен (последний, в данном случае) в буфер
4. git revert commit_id
5. Меняем заголовок в редакторе, если нужно. Сохраняем.
6. git commit
7. git push origin
