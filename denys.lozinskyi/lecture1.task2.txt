условие1. У вас в индексе(стейдж области) уже есть много добавленных файлов, которые вы не хотите потерять
"""


ВАРИАНТ РЕШЕНИЯ:
Если те файлы, которые лежат на стейдже, мы не хотим добавлять в коммит в числе исправлений, которые мы хотим привнести,
для их сохранения применим git stash:

1. git stash save
2. Делаем исправления, которыми мы хотим проамендить последний коммит. Потом:
3. git add . (или те файлы, которые мы хотим добавить)
4. git commit --amend
5. В редакторе меняем название коммита. Сохраняемся.
6. Если название менять не нужно, можно воспользоваться git commit --amend --no-edit
7. git stash pop (если нужно вернуть застэшенные файлы назад в ветку)
